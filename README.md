AeOrganizer is an After Effects script that automatically organizes your project into folders based on media type and file extension.

## Installation
Download `AeOrganizer.jsx` and copy it into the After Effects Scripts folder.
Close Adobe After Effect.

**Windows**  
`C:\Program Files\Adobe\Adobe After Effects\Support Files\Scripts\`

**macOS**  
`Applications/Adobe After Effects/Scripts/`

## Usage
Open an After Effects project, then go to  
`File > Window > AeOrganizer`  
Click **Organize Project** to sort the project automatically. Assets are organized into folders such as Videos, Images, Audio, Compositions, Solids, with subfolders by file extension.

## Undo
Click **Undo** to revert the organization in one step, or use `Ctrl + Z / Cmd + Z`.

## Compatibility
Adobe After Effects CC and later. Supports videos, images, audio, 3D models, and compositions.

## Notes
Unused compositions are moved to a dedicated folder. Source files are not modified, only the project structure.
